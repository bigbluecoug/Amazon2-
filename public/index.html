Account setup
  1. New accounts are stored securely on the server with hashed passwords.
  2. The first created account becomes the gift ideas admin.
  3. Set ALLOW_ACCOUNT_REGISTRATION=false after creating the accounts you want.
  4. Keep a long random SESSION_SECRET set on Forge.

Add people one by one or paste a CSV, then mark each prospect ready once the shipping address is complete.

Confirm the sequence to lock the gifts and messages, then process due gifts when you are ready to create order records.

Open automation console

An admin connects Amazon once. After that, team members can run the send queue without handling API settings.

Advanced Amazon admin settings
01 Admin-only setup happens outside the team workflow.

One workspace admin adds the Amazon Business app values to Forge. Team members should not need these fields during daily use.

02 Connect through the guided console.

Use the automation console to approve Amazon Business access. GiftFlow fills the private connection values after Amazon returns.

03 Keep live sending behind review.

GiftFlow requires sequence confirmation and ready recipients before gifts can be queued for the live connector.

04 Use test mode for dry runs.

Choose Test run only when you want to validate the flow without treating records as ready for the live connector.
